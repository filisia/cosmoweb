Bluetooth Low Energy (BLE) Browser Support Guide

This guide provides information on browser support for Bluetooth Low Energy (BLE) and instructions for enabling BLE in supported browsers.

Google Chrome

Versions 56 to 122 support BLE. To use BLE, ensure that you have granted the necessary permissions to the website to access Bluetooth devices.

Microsoft Edge

Versions 79 to 119 support BLE. Like Chrome, users must grant permission to the website to access Bluetooth devices.

Opera

Versions 43 to 104 support BLE. Users need to enable Bluetooth access for the website when prompted.

Samsung Internet Browser

Versions 6.2 to 23 support BLE. Ensure that the browser has the necessary permissions to use Bluetooth.

Opera Mobile

Version 73 supports BLE. Grant the necessary permissions to the website for Bluetooth access.

UC Browser for Android

Version 15.5 supports BLE. Make sure to allow Bluetooth access when prompted by the browser.

Android Browser

Version 119 supports BLE. Users should grant the required permissions for Bluetooth access.

Baidu Browser

Version 13.18 supports BLE. Ensure that Bluetooth permissions are granted when requested by the browser.

Browsers with No Support for BLE

  • Safari (including on iOS)
  • Firefox
  • Internet Explorer
  • Opera Mini

These browsers do not currently support BLE. Please use a supported browser.
